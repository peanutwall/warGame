代码主要参数解释：
mapSize int 游戏地图的大小
isBoundary bool  表示是否有边界
isEven bool 表示地图是否有偶数land
isSymmetrical bool 表示地图是否对称

游戏说明：
下方区域为游戏区，第一排为战场（warfield），若为金色则该区域未被占领或平局，为红或蓝色则为a或b占领
第二三排为策略区
点击策略区的圆圈，若圆圈变绿则点击正确，输入在该区域放置的兵力，按空格完成；
完成所有兵力的布置之后，按回车，进行兵力比较，并在warfield上显示结果；兵力总和必须小于initialsoldiers，
否则无法进行比较。
